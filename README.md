# Testblatt

zum Üben der Testblätter der Abzeichen der Niederösterreichischen Feuerwehrjugend.
Die Erprobung bezieht sich auf die eigene Feuerwehr, in diesem Fall auf die Freiwillige Feuerwehr Kilb, wie auch bei dem Gerätequiz das verlinkt ist.

- Fertigkeitsabzeichen
- Wissenstest
